## 핵심 메시지

**병렬 작업의 전제 = 계약 우선 확정 (M0)**
- 01 API 계약 + 02 공유 스키마 + 03 Java 인터페이스를 전원이 합의하면,
- 5명이 서로 기다리지 않고 독립적으로 개발 가능

**병렬 가능 장치**
- 프론트: MSW(Mock Service Worker)로 백엔드 없이 개발
- 백엔드 B: MockLlmClient로 AWS Bedrock 없이 개발
- 백엔드 B/C: 백엔드 A 인터페이스 stub으로 선행 개발
- 모듈 경계 + 브랜치 전략으로 코드 충돌 방지
