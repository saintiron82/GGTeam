페이지 전환 기본 원칙
Slide 방향 원칙: 계층이 깊어질수록(목록→상세) 우측 또는 아래로, 복귀 시 반대 방향. 수직 이동(translateY)은 ±8px 이하로 제한 — 그 이상은 현기증 유발.
중장년층 사용자 고려: 빠른 슬라이드나 큰 이동 거리는 멀미감 유발 가능. 이동 거리 최소화 + Fade 병행이 가장 안전.
prefers-reduced-motion 대응: @media (prefers-reduced-motion: reduce) 시 모든 transform 제거, duration을 1ms로 단축. 접근성 필수 대응.
CSS 구현 예시
:root {
--dur-fast: 100ms;
--dur-base: 200ms;
--dur-slow: 300ms;
--ease-out: cubic-bezier(0, 0, 0.58, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
}
.page-enter { opacity: 0; transform: translateY(-6px); }
.page-enter-active { opacity: 1; transform: translateY(0);
transition: opacity var(--dur-base) var(--ease-out), transform var(--dur-base) var(--ease-out); }
@media (prefers-reduced-motion: reduce) {
*, *::before, *::after { transition-duration: 1ms !important; }
}

Figma Smart Animate 사용 팁
팁상세
Layer 이름 일치 필수Smart Animate는 동일한 레이어 이름을 추적. 버튼 label이 "Label"로 통일되어야 hover 시 텍스트 변화가 감지됨
Easing 근사값 사용ease-out → "Ease Out", ease-in-out → "Ease In and Out" 선택. Custom은 미지원이므로 가장 유사한 Figma 기본값 선택
Spring 대안spring 효과는 Figma에서 "Spring" easing 선택 후 Stiffness/Damping 조정 (Stiffness 500, Damping 30 권장)
Overlay 방식 ModalModal은 Prototype에서 "Open overlay" 사용. Position: Centered, Background: #242021 40% opacity
Sidebar Frame 분리Expanded Frame과 Collapsed Frame을 별도로 만들고, 토글 버튼에 "Navigate to" + Smart Animate 연결
Duration 통일Figma Prototype duration은 ms 단위 직접 입력 가능 (200, 300 등). 토큰값과 일치시킬 것
