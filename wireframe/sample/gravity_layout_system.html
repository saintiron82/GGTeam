BreakpointPC / Tablet / Mobile레이아웃 Frame별 적용
Figma Variables의 Mode 스위치를 활용해 동일한 컴포넌트 인스턴스에서 라이트/다크, PC/모바일 스펙 전환이 가능하도록 구성한다. Pages 섹션의 각 화면 Frame에 breakpoint Mode를 지정하면 레이아웃 스펙이 자동 적용된다.
